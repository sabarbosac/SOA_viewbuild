    How to use the Viewbuild

    View Build is a web app that lets youuse the viewshed function in New York city it has added the database of the different heights and ground elevations of buildings. The app is pretty intuitive if you follow the next instructions that are depicted in the picture below

      1. Select a Base Point

    • The Base poin is the point in which the user is going to set the bench mark to see if the view points are avaible in the sight of the base point. Viewshed analysis could estimate the impact of the addition of a large building. The viewshed analysis would show all the areas from which the building could be seen as well as any views that would be obscured from any particular location.

      • 2. Select the view Points

    • The points that can be viewed from the base point are the viewing points in the Viewshed tool. Consider the point that you want to be viewed from the base point that you have selected in the city. Without the viewing point, the tool doesn't run. The minimum number of viewing points required to run the tool is 2 points, so the observer can be viewed from the other points marked.

      • 3. Select a blocking Point

    • A blocking point is just another view point in the way of the view results of the existing base and view points. Its optional and is done only for a few purposes.

      • 4. Download or Upload

    • Select Download to copy the data in your computer. Select Upload to share the data with HydroShare.
